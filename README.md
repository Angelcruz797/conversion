# Projecto hecho por el equipo de NovaSis

## Como inicializar el proyecto

requisito obligatorio tener node intalado (https://nodejs.org/es)

1. Descargar el repositorio
2. ejecutar en la terminal ```npm install``` para instalar todas las dependencias del proyecto
3. ejecutar en la terminal ```npm run dev``` para ejecutar el proyecto
